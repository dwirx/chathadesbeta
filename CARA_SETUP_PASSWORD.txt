================================================================================
CARA SETUP PASSWORD UNTUK HADESCHAT
================================================================================

LANGKAH 1: BUKA FILE .env
--------------------------
Di root folder project (hadeschat/), buka atau buat file bernama: .env

LANGKAH 2: TAMBAHKAN PASSWORD
------------------------------
Tambahkan baris ini ke file .env:

VITE_APP_PASSWORD=test123

** GANTI "test123" dengan password pilihan Anda **

LANGKAH 3: CONTOH FILE .env LENGKAP
------------------------------------
# Password Protection
VITE_APP_PASSWORD=test123

# API Keys (minimal 1 diperlukan)
VITE_GROQ_API_KEY=your_groq_api_key_here
VITE_OPENROUTER_API_KEY=your_openrouter_api_key_here
VITE_TOGETHER_API_KEY=your_together_api_key_here
VITE_POE_API_KEY=your_poe_api_key_here

# Optional
VITE_SITE_URL=http://localhost:5173
VITE_SITE_NAME=HadesChat

LANGKAH 4: RESTART SERVER
--------------------------
Setelah menyimpan file .env, restart development server:

1. Tekan Ctrl+C untuk stop server
2. Jalankan: npm run dev
3. Buka browser: http://localhost:5173

LANGKAH 5: LOGIN
-----------------
1. Masukkan password yang Anda set di .env
2. Klik tombol "Masuk"
3. Selesai! Anda bisa menggunakan aplikasi

================================================================================
CATATAN PENTING

✓ Session bertahan sampai browser tab ditutup
✓ Pastikan file bernama ".env" (dengan titik di depan)

================================================================================
TROUBLESHOOTING
================================================================================

Problem: Password tidak berubah
Solusi: Restart server dan refresh browser (Ctrl+Shift+R)

Problem: File .env tidak terbaca
Solusi: Pastikan file di root folder dan bernama ".env" (bukan "env.txt")

Problem: Selalu diminta login
Solusi: Pastikan browser tidak dalam mode Incognito/Private

================================================================================
UNTUK PRODUKSI (DEPLOYMENT)
================================================================================

VERCEL:
1. Go to: Settings → Environment Variables
2. Tambahkan: VITE_APP_PASSWORD = your_strong_password
3. Save dan redeploy

NETLIFY:
1. Go to: Site settings → Environment variables
2. Tambahkan: VITE_APP_PASSWORD = your_strong_password
3. Save dan redeploy

CUSTOM SERVER:
1. SSH ke server
2. Edit file .env di server
3. Tambahkan: VITE_APP_PASSWORD=your_strong_password
4. Rebuild: npm run build
5. Restart service
